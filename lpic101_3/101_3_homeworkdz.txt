Понравилась таблица с уровни запуска линукс
На Ubuntu 20.04 можно запустить команду runlevel с нужной вам цифрой - это первый способ
Второй sudo systemctl enable multi-user.target
sudo systemctl set-default multi-user.target

2) Переключиться между несколькими режимами работы.
На сколько я понимаю автор имеет ввиду переключение между рабочими столами tty
Это реализует через комбинацию CTRL+ALT+ F1 b так далее
Если имеется ввиду между уровнями работы системы то как описано выше с помощью команды runlevel

3) Убрать из автозапуска по умолчанию службу сети.
Сначала остановим сервис с помощью команды systemctl stop networks
Далее удалим его с автозагрузки нашей системы systemctl disable networks

4) Проверить статус службы аудио.
Через systemctl status <имя сервиса>
Через ssytemctl list-units --state=running(stop,dead)

5) Перезагрузить ПК через работу с уровнями выполнения или таргетами.
На Ubuntu 20.04 можно запустить команду runlevel 6
Или sudo systemctl isolate reboot.target
